# 7. Utilities


> We propose this nomenclature of prefixes: 
> 
> - `as-`: generic graphical form ;
> - `of-`: group of elements ;
> - `is-` interactive state of the system ; 
> - `on-`: user interaction ;
> - `in-`: internal concept ;
> - `with-`: functional rule ; 
> - `from-*-to-`: adaptive patterns ;
